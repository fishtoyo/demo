多行文字刪節號
搭配jQuery則可將刪節號用於多行文字
此範例需要使用 ellipsis.js 文件。
Demo
多行文字刪節號樣式測試123多行文字刪節號樣式測試456多行文字刪節號樣式測試789多行文字刪節號樣式測試000多行文字刪節號樣式測試111
Script
<script src="jquery.js"></script> <!-- jQuery 主程式 -->
<script src="ellipsis.js"></script>

<script>
jQuery(document).ready(function() {
// Multi-line Ellipsis
jQuery('.ellipsis').ellipsis({
height: 90 // 指定區塊內容高度
});
});
</script>
CSS
.ellipsis {
width: 200px; // 寬度200px
overflow: hidden; // 若內容溢出指定的寬/高，則隱藏溢出的內容
}
Body 以下程式碼放置於 <body>...</body>
<div class="ellipsis multiline">
多行文字刪節號樣式測試123多行文字刪節號樣式測試456多行文字刪節號樣式測試789多行文字刪節號樣式測試000多行文字刪節號樣式測試111
</div>
寬度設定
若不指定寬度，則會以父層區塊的寬度為基準
