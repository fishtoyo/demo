刪節號樣式 - 單行
純CSS寫法
單行文字要進行刪節號樣式時，使用CSS的 white-space 跟 text-overflow 屬性，搭配 overflow 即可。
CSS
.single_line {
width: 200px; // 指定寬度為200px
overflow: hidden; // 若內容溢出指定的寬/高，則隱藏溢出的內容
white-space: nowrap;  // 文字不換行
text-overflow: ellipsis; // 溢出時的樣式設定為 ellipsis (...)
}
Body 以下程式碼放置於 <body>...</body>
<div class="single_line">
單行文字刪節號樣式測試123單行文字刪節號樣式測試456單行文字刪節號樣式測試123單行文字刪節號樣式測試789
</div>
