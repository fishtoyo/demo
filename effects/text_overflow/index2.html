刪節號樣式 - 多行
搭配jQuery則可將刪節號用於文字多行
此範例需要使用 ellipsis.js 文件。
引入檔案
<script src="jquery.js"></script> <!-- jQuery 主程式 -->
<script src="ellipsis.js"></script>
jQuery
jQuery('.ellipsis').ellipsis();
CSS
.ellipsis {
width: 100px;      // 設定一個區塊寬度為100px
height: 90px;      // 設定一個區塊寬度為100px
overflow: hidden; // 若內容溢出指定的寬/高，則隱藏溢出的內容
}
Body 以下程式碼放置於 <body>...</body>
<div class="ellipsis multiline">
多行文字刪節號樣式測試123多行文字刪節號樣式測試456多行文字刪節號樣式測試123多行文字刪節號樣式測試789
</div>
